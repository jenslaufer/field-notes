# Stop Vibe Coding. Start Getting Customers.

> Building is now free — **200,000 vibe-coding projects launch on Lovable every
> single day** — so the moat moved to distribution. Greg's thesis: marketers will
> be the wealthiest people of the next 10 years. He hands you **7 distribution
> weapons you can start this week**. The meta-rule: **distribution first, product
> second, always.**

## The mindset flip

- **Hierarchy inverted.** 2014 SV ranked engineers > product > marketers. Now
  it's **distribution > product > developers**. Code is fully commoditized.
- **The trap.** Build → launch → silence → build more features → launch → silence.
  "If you build it they will come" is false.
- **What smart builders do.** Grow an audience to ~1,000, ask what they need,
  build it in a weekend, launch to a *warm* audience, iterate, monetize.
- **Proof point.** Peter Levels: $3M+ revenue, 0 employees, ~750K followers,
  ~125K tweets in 8 years. The audience + SEO is the moat, not the code.

## The 7 strategies

### 1. MCP server as your sales team
- **Zero-CAC discovery.** User asks an LLM a question → AI discovers your MCP
  server → returns your product. The AI assistant *is* your sales team, 24/7.
- **Building an MCP in 2026 = building for mobile in 2010.** Early movers own the
  LLM-native channel. A friend in fintech got **150+ installs in 30 days, $0 ad spend.**
- **This week:** (1) name the question your product answers, (2) vibe-code an MCP
  that returns that data (~24h), (3) publish to **Smithery, MCPT, Open Tools**.

### 2. Programmatic SEO — 10,000 pages in a weekend
- **Pattern:** "best X for Y" (best CRMs for dentists, roofing in Miami).
- **Pipeline:** scrape structured data with **Firecrawl** → Next.js/Webflow/WordPress
  template → AI-generate *unique* paragraphs per page (not variable swaps) →
  human-in-the-loop edit so it doesn't read as AI.
- **The math:** 10,000 pages × 30 visits/mo = 300K visitors → 2% conv = 6,000
  sales → $10 each = **$60K/mo from pages built once.** SEO compounds slowly.
- **This week:** publish 100 pages as MVP, monitor indexation, then scale.

### 3. Free tool as top of funnel — "the tool is the marketing"
- **Model (Ahrefs backlink checker):** give a taste, hook on the upsell.
- **Viral loop:** build tool → user gets instant value/score → captures email →
  user shares score → more discovery → upsell to paid.
- **Why now:** Claude Code makes a free tool a *day's* work, not months. Greg runs
  a "free-tool calendar" the way others run a content calendar.
- **This week:** ask your LLM "give me 10 free-tool ideas as top-of-funnel," pick
  the best, ship by lunch.

### 4. Answer Engine Optimization (AEO) — be the source AI cites
- **AEO in 2026 = SEO in 2010.** First movers own niches for years; zero-click is
  growing. Peter Levels saw **AI referrals jump 4% → 20% in one month.**
- **Tactics:** structured direct answers, FAQ format, schema markup, comparison
  tables AI can parse. Citation-worthy, not 3,000-word fluff.
- **This week:** Google your customer's top 20 questions, write definitive
  answers, add schema/FAQ blocks, publish on an authority domain, monitor
  citations with **Otterly / Profound / manual testing.**

### 5. Viral artifacts — make your output shareable
- **The artifact = what your user wants to brag about.** Spotify Wrapped gets
  ~100M shares each December. GitHub contribution graph, Stripe Atlas
  incorporation milestone, Duolingo streak.
- **This week:** (1) find the milestone they'd screenshot, (2) make it beautiful,
  branded, *subtle* logo (no one shares a giant logo), (3) add a share button that
  pre-fills the post. Works in B2B too — people share in Slack/Teams.

### 6. Buy a niche newsletter — skip the 0-subscriber problem
- **Inherit trust on day one.** Buy a **10,000-sub niche newsletter for $5K–$20K.**
  Many owners make $0–$500/mo and would take a fair offer.
- **You own the channel** — a direct line, not a suppressible social account.
- **This week:** browse **Duuce.com / Newsletter Investor**, or search the niche on
  X/Substack; target 5K–50K subs; DM the owner "ever thought about selling?"

### 7. AI repurposing engine — 1 pillar, 7 channels
- **One hero piece** (podcast, video, essay) → 5–10 tweets, 3–5 LinkedIn posts,
  2–3 short videos, 1 newsletter, 5–10 quote graphics, email sequences.
- **Easiest input:** record a 30-min voice memo or internal video — easier than
  writing a 3,000-word essay; the value is in the repurposing.
- **Tools:** Claude Code + a **Remotion skill** for short video; OpenClaude / Claude
  dispatch / Claude co-work / Perplexity computer to automate. Avoid slop with a
  strong brand + reference images.

## Marketing tips

- **Distribution first, product second** — grow ~1,000 fans, ask, build in a
  weekend, launch warm.
- **Programmatic SEO math to copy:** 10K pages × 30 visits × 2% × $10 = $60K/mo.
- **Newsletter acquisition:** $5K–$20K buys a 10K-sub direct channel today.
- **Free-tool calendar** beats a content calendar — ship one tool/week.
- **Shots on net:** you don't need many followers; with For You pages, you never
  know when a niche post goes viral — maximize volume.

## Why this matters for us

- **Reinforces the launch-kit distribution-first thesis** — validate offer×audience
  on a warm list before building, not after.
- **Free tool + email capture + viral artifact** is a directly copyable lead-funnel
  recipe for any launch-kit product.
- **Programmatic SEO + AEO** are concrete, automatable channels we can stand up with
  Firecrawl + Claude Code.
- **Newsletter acquisition** is a cheat code for cold-start distribution worth
  pricing against LTV.
