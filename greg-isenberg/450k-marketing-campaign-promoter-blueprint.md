# Claude Code Built My $450K Marketing Campaign

> You vibe-coded a thing and launched to crickets. Jonathan's fix: stop treating
> "build cool AI systems" as the job. **A CEO's actual job is to promote.** He
> shows the **Promoter Blueprint** — a 4-stage selling machine (traffic → holding
> pattern → selling event → conversion) — then uses Claude/Claude Code only as
> *support*. Live, he's mid-campaign expecting **~$450K this week** (4,000 webinar
> sign-ups; typical range $250K–$500K).

## The CEO reframe (the foundation, no AI)
- **Promotion is the job.** Anthropic's Dario at Davos, Sam Altman, Peter Levels —
  ~50% of their job is promoting their business. You know their names *because*
  they're always out there.
- **The procrastination trap.** AI tools used wrong become a "procrastination
  station" — you spend 5 days building systems and make $0 more.
- **Restaurant analogy:** perfect automated kitchen, never told a single person it
  exists → it dies. Build ≠ promote.
- **Use AI ~20% of the time** Greg's audience does; spend the rest making money,
  getting users, closing deals.

## The Promoter Blueprint (4 stages)
### 1. Traffic — organic or paid (the only two)
- **Organic:** other people's podcasts, events/networking, free social content.
- **Paid:** Meta, TikTok, YouTube ads.
- **Rule:** never push traffic straight at "buy my product." Push into a *holding
  pattern.*

### 2. Holding pattern — warm them up
- **Where:** email newsletter (his biggest), podcast, YouTube, X posts — your daily/
  weekly content. Keep attention on you; don't pitch hard here.
- **Most people are decent at stages 1–2** (followers, engagement) and **terrible at
  stage 3.**

### 3. Selling event — the bridge from attention to money
- **Webinar / live demo / workshop** (Superhuman-style; no hard pitch needed —
  seeing the product *is* the pitch).
- **Email campaign** — a string of 3–4 emails leading holding-pattern people to buy.
- **Retargeting** — upload your email list to Meta, build a lookalike, retarget.
- **Manual outreach** — scan new sign-ups; a microsoft.com email → book an
  enterprise call.

### 4. Conversion — and recycle the rest
- Non-converters (most) fall **back into the holding pattern** until the next event.
- **The math he runs:** +5,000 newsletter subs → ~200 more webinar attendees → ~12
  more buyers. Traffic is the oxygen; no traffic in = empty events = no sales.

## How Claude supports each stage (his actual workflow)
- **Brain-dump → asset.** Recorded a 15-min voice memo (WhisperFlow), dropped it in
  Claude, got an **ADHD-friendly scannable HTML doc** to read live while talking.
- **Self-teaching instructions.** After each chat: "give me a pasteable set of
  instructions to add to this project" — the project gets smarter over time.
- **Persistent research file.** A separate chat researched Greg's X/YouTube once,
  saved to a file, so context isn't rebuilt every chat.
- **Claude → Claude Code handoff.** When the HTML blueprint was good: "create a
  CLAUDE.md with full project context," export the HTML, move to Claude Code,
  build it, push to GitHub → Vercel. **~1 hour, mostly in the background.**
- **The marketing-copy machine.** A Claude project pre-loaded with his books + a
  **17,766-line doc of other people's marketing emails**, distilled into the
  CLAUDE.md, generates email/webinar copy. He prefers the Claude *app* UI for pure
  marketing (likes the overview), Claude Code for building.
- **Ask-user-question / plan mode** to *grill himself* — a 30-min interrogation on
  webinar structure produces better output than a mega-prompt.

## Marketing tips
- **Lead-magnet generation, live:** fed the episode context + CLAUDE.md, asked for
  3 lead-magnet ideas → "Promoter Blueprint one-pager," "50 prompts for the
  promoter CEO," **"Cave Dweller Audit: are you building or promoting?"** (a
  provocative self-assessment that exposes the builder trap). Then Claude Code
  builds the landing page.
- **Custom-feeling campaigns at speed:** a full lead-magnet campaign that took 2–3
  days now takes ~half a day — make each one feel bespoke to the content.
- **Scale, don't optimize.** The play right now is abundance: keep your small team,
  triple its output, run **5 campaigns/month instead of 1**, ship many ad
  variations instead of refreshing every 6 weeks.
- **Don't over-build.** Laura spent 3 days building a proposal tool; Claude then
  named an off-the-shelf option that worked day one. Ask first.

## Why this matters for us
- **The Promoter Blueprint is a clean funnel template** — traffic → holding pattern
  → selling event → recycle — directly mappable to launch-kit's lead-funnel +
  email thesis.
- **The recycle insight** (non-buyers return to the holding pattern) argues for a
  durable newsletter as the core owned asset, not one-shot launches.
- **CLAUDE.md as accumulating brand/marketing memory** (books + swipe files) is a
  pattern we can adopt for repeatable on-brand copy.
- **"Less prep, more action"** + "scale over efficiency" is the right posture for
  shipping marketable features fast.
